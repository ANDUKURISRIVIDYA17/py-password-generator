# Python Password Generator

A lightweight, secure command-line tool written in Python that generates customizable, strong passwords to help users maintain robust digital security.

## 🚀 Features
* Generates completely randomized, highly secure passwords.
* Customizable options for length and character types (uppercase, numbers, symbols).
* Standard alphanumeric and punctuation support.

## 🛠️ Tech Stack
* **Language:** Python
* **Modules:** `random`, `string`
